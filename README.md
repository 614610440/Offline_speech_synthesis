# Offline_speech_synthesis

## 离线语音合成的简单方法
> 目前支持中文和阿拉伯数字  
> #测试版本， 请使用tag最新版

## 开发环境
> ubantu18 + python3.6.7  

## 依赖安装
> ```` 
> pip3 install -r requirment.txt   
> ```` 

## 使用
> ````   
> 
> mkdir cache
> python3 demo.py   
> ````  

## 说明
> 大更新!!! 更新线程方式, 只用一个线程, 节省资源, 提高灵活性.  
> wav为处理过后的音频,更换音频可以提升合成效果。
