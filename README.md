# Offline_speech_synthesis

#### 离线语音合成的简单方法
> 目前支持中文和阿拉伯数字  
> #测试版本， 请使用tag最新版

#### 开发环境
> ubantu18 + python3.6.7  

#### 依赖安装
> ```` 
> sudo apt-get install ffmpeg  
> pip3 install -r requirment.txt   
> ```` 
#### 使用
> ````   
> 
> python3 demo.py   
> ````  

#### 说明
> voice 文件用于测试，有多种发音，可以替换voice文件来提升合成语音效果。
